### 1.2 규칙 타입 상세 비교

#### Cursor (4가지 타입)
| 타입 | 설명 | 적용 조건 |
|------|------|----------|
| **Always** | 모든 세션에 적용 | `alwaysApply: true` |
| **Auto Attached** | 파일 패턴 매칭 시 | `globs: ["**/*.tsx"]` |
| **Agent Requested** | AI 판단 시 적용 | `description` 기반 |
| **Manual** | 수동 호출 시 | `@규칙명` 멘션 |

#### Claude Code (2가지 타입)
| 타입 | 설명 | 적용 조건 |
|------|------|----------|
| **Global** | 모든 파일에 적용 | `paths` 없음 |
| **Path-specific** | 경로 매칭 시 적용 | `paths: ["src/api/**"]` |

#### Gemini CLI (모드 기반)
| 모드 | 설명 | 트리거 |
|------|------|--------|
| **Default** | 기본 대화 | 기본 상태 |
| **Explain** | 설명/조사 | `<explain>` 태그 |
| **Plan** | 계획 수립 | `<plan>` 태그 |
| **Implement** | 구현 실행 | `<implement>` 태그 |
