## 🔄 의도 파악 우선순위

1. **명시적 서버명**: "크롬으로", "구글독스로" → 직접 매핑
2. **의도 키워드**: "이메일 보내줘" → Gmail
3. **맥락 파악**: "최신 정보" → Exa, "최신 문서" → Context7
4. **복합 의도**: 여러 키워드 조합 분석

- 서버명이 명시되면 해당 서버 사용
