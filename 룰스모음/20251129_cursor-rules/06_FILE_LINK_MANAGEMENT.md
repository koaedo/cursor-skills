# 파일 링크 종속관계 관리 룰스

**독립 모듈**: 이 룰스는 독립적으로 유저룰스에 붙여넣을 수 있습니다. 파일 종속성 없음.

## 🎯 목적

**파일 링크 종속관계 관리**: 생성되는 파일이 삭제되지 않으므로, 로그와 보고서에 파일 링크를 추가하여 종속관계를 만들어 맥락관리, 환각관리, 버전관리를 가능하게 함

---

## 📋 핵심 원칙

**생성되는 파일이 삭제되지 않으므로, 로그와 보고서에 파일 링크를 추가하여 종속관계를 만든다**

**장점**:

1. **맥락관리**: 보고서/로그가 어떤 파일들을 참조했는지 추적 가능
2. **환각관리**: 보고서/로그가 실제 파일을 참조하므로 검증 가능
3. **버전관리**: 파일 버전과 보고서/로그 버전을 연결하여 버전 추적 가능
4. **시계열 추적**: 시간 순서대로 파일 변경사항과 보고서/로그의 관계 추적 가능

---

## 📋 보고서 파일 링크 규칙

### 자동 수집 대상

- 보고서 작성에 참조된 파일
- 보고서 내용과 관련된 파일
- 보고서 분석 대상 파일

### 링크 형식

```markdown
## 📎 관련 파일 (종속관계)
- [파일명](경로) - 역할: [파일의 역할 설명]

## 📋 참조 보고서
- [보고서명](경로) - 생성시각: YYYYMMDD_HHMMSS

## 🔗 버전 정보
- 보고서 버전: YYYYMMDD_HHMMSS
- 참조 파일 버전: [파일명] - YYYYMMDD_HHMMSS
```

### 자동화 프로세스

```
보고서 생성 시:
  1. 보고서 작성에 참조된 파일 목록 수집
  2. 각 파일의 경로 및 버전 정보 확인
  3. 파일 링크 섹션 자동 생성
  4. 보고서에 파일 링크 섹션 추가
  5. 버전 정보 섹션 추가
```

## 📋 LOG 파일 링크 규칙

### 자동 수집 대상

- 트리거 기간 동안 변경된 파일
- 트리거 기간 동안 생성된 보고서
- 트리거와 관련된 파일

### 링크 형식

```markdown
## 📎 관련 파일 (트리거 기간 변경)
- [파일명](경로) - 변경시각: YYYYMMDD_HHMMSS

## 📋 관련 보고서 (트리거 기간 생성)
- [보고서명](경로) - 생성시각: YYYYMMDD_HHMMSS

## 🔗 버전 정보
- LOG 버전: YYYYMMDD_HHMMSS
- 트리거 기간: YYYY-MM-DD HH:MM ~ YYYY-MM-DD HH:MM
- 참조 파일 버전: [파일명] - YYYYMMDD_HHMMSS
```

### 자동화 프로세스

```
LOG 생성 시:
  1. 트리거 기간 동안 변경된 파일 목록 수집
  2. 트리거 기간 동안 생성된 보고서 목록 수집
  3. 각 파일/보고서의 경로 및 버전 정보 확인
  4. 파일 링크 섹션 자동 생성
  5. LOG에 파일 링크 섹션 추가
  6. 버전 정보 섹션 추가
```

## 📋 관리 시스템

### 맥락관리 (Context Management)

**목적**: 보고서/로그가 어떤 파일들을 참조했는지 추적
**활용**: 역방향 추적, 순방향 추적, 관계 분석
**자동화**: 파일 수정 시 관련 보고서/로그 자동 검색, 보고서/로그 생성 시 관련 파일 자동 수집

### 환각관리 (Hallucination Management)

**목적**: 보고서/로그가 실제 파일을 참조하므로 검증 가능
**활용**: 파일 존재 확인, 내용 검증, 버전 일치 확인
**자동화**: 보고서/로그 검증 시 참조된 파일 존재 여부 확인, 내용 일치 여부 확인

### 버전관리 (Version Management)

**목적**: 파일 버전과 보고서/로그 버전을 연결하여 버전 추적
**활용**: 버전 추적, 버전 일치 확인, 버전 롤백
**자동화**: 파일 수정 시 새 버전 정보 추가, 보고서/로그 생성 시 참조된 파일 버전 정보 수집

### 시계열 추적 (Time Series Tracking)

**목적**: 시간 순서대로 파일 변경사항과 보고서/로그의 관계 추적
**활용**: 시간 순서 추적, 트리거 분석, 패턴 분석
**자동화**: LOG 생성 시 트리거 기간 동안 모든 파일 변경사항 수집, 보고서 생성 시 작성 시점의 파일 버전 정보 수집
